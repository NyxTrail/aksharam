Aksharam is an easy to use reference app to learn Indian language scripts by comparing the alphabets. Know the script for an Indian language? See what those letters look like in a different Indian language.

<b>Features:</b>
<ul>
<li>Tap a letter to see what it corresponds to in a different language</li>
<li>Long tap a letter to see more detailed information about a particular letter</li>
<li>Includes easy to visualise comparison of vowels, consonants and joint letters</li>
<li>See how letters combine with each other (consonants + vowels, consonants + consonants)</li>
<li>Transliteration of text into a script you can understand</li>
<li>Practice your ability to recognize the script for a language by typing in corresponding letters in a language you are familiar with</li>
</ul>

<b>Note:</b>
<ul>
<li>This app is designed for those who already know the script for an Indian language.</li>
<li>Currently Kannada, Hindi and Malayalam languages are supported.</li>
<li>UI language is in English.<li>
</ul>
